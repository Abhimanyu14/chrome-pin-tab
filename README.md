# Tab Pin Toggler

Chrome extension to pin a tab using keyboard shortcut.

## Keyboard shortcut

<kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>X</kbd> or <kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>X</kbd> for Mac
